# Installation

## Manual installation:
### Upload the "ohio" folder to the "/wp-content/themes/" directory;

# Activiation and Usage

## 1. Activate the theme through the "Appearance > Themes" menu in WordPress;
## 2. Activate the theme license through the "Ohio Theme > Dashboard" menu in WordPress;
## 3. Check Theme Settings menu to change the appearance using pre-made options;

# Custom Logo

This theme supports custom logo for the header part;
You can change or remove it from "Ohio Theme > Theme Settings > General";
